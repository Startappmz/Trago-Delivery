# Trago Delivery — Ultra Compact UI

Aplicado tratamento visual mais compacto sobre a versão “Mapa Realtime + Interações”.

## Painel Admin
- Header mobile mais baixo.
- Bottom navigation mais curta.
- Botões, textos, inputs e ícones reduzidos.
- Cards estatísticos mais densos.
- Mapa realtime mais compacto, com HUD e lista de motoristas reduzidos.
- Popups do mapa mais pequenos.
- Tabelas em modo mobile mais apertadas.

## Painel Motorista
- Header, logotipo, status GPS e acções mais compactos.
- Bottom navigation menor.
- Cards de entrega mais densos.
- Detalhe de entrega, mapa, HUD e botões de rota mais curtos.
- Ganhos, formulários e configurações com menos espaçamento.

## Segurança da alteração
- Sem alterações ao backend.
- Sem alterações aos endpoints.
- Sem alterações à autenticação.
- Sem alterações à base de dados.
- Alteração focada em HTML textual e CSS visual.
