# Trago Delivery — Refinamento compacto e mapa curto

Aplicado sobre a versão Ultra Compact UI.

## Ajustes feitos

- Redução equilibrada dos textos que ainda pareciam grandes.
- Barra inferior do admin corrigida usando a classe real `.admin-bottom-item`.
- Barra inferior do motorista reduzida sem cortar labels.
- Botões com altura menor, mas com line-height e overflow corrigidos para evitar cortes visuais.
- Mapa realtime do admin reduzido para facilitar o scroll.
- Mapa do motorista reduzido e HUD reorganizado para ocupar menos espaço.
- Estatísticas do mapa mais compactas.
- Lista de motoristas no mapa com altura menor.

## Escopo

Não houve alteração em backend, endpoints, autenticação, Supabase ou contratos de API.
